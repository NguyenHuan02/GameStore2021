       <li>1. Install <code>JDK 1.8+</code> (if not installed)</li>
       <li>2. Install <code>SQL Server 2008+</code> (if not installed)</li>
       <li>3. Account MS SQL : change account <code>sa</code> with pasword <code>''</code></li>
